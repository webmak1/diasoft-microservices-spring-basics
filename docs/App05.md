# 05. Создание и обновление структуры базы данных

<br/>

Подключить в микросервис работающий со Spring Data JPA создание структуры БД с помощью liquibase. Скрипты должны создать структуру таблиц в СУБД Postgres.




<br/>

### Запуск приложения

<br/>

```
$ cd apps/app05
```

<br/>

```

```
